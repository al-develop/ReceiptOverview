# Receipt Overview

A little piece of software, to digitalize Receipts and export them as a .csv

## Made with

- Avalonia UI 11.0.X
- C# & dotnet 7.0
- Icons8

## Build for

- Fedora Linux
- Windows 10

(not tested on other systems, but should build and run wherever the dotnet 7.0 Runtime is installed)
